\# 🎯 ProzorroHunter



Автоматичний моніторинг тендерів Prozorro 24/7 з красивим веб-інтерфейсом.



\## Особливості



\- ✅ Glassmorphism дизайн

\- ✅ Реальний моніторинг Prozorro API

\- ✅ SQLite база даних

\- ✅ Безкоштовний деплой на Render



\## Запуск локально

```bash

pip install -r requirements.txt

python backend.py

```



Відкрий: http://localhost:10000
